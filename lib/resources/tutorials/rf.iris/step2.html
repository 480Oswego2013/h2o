The random forest algorithm uses a training datset to build a model. In this tutorial, we will use the common Iris dataset for forest building.

The iris dataset is provided here. Please, download it and save it on your disk.

To upload the dataset into H2O application, please, use the menu option Data > Upload or direct link.

If connected data can be imported from URL directly into H2O application, please, use the menu option Data > Import URL or direct link.
