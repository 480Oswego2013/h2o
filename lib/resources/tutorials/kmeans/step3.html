The Inspect page showed after a successful parse has a direct K-means link.

You can also use the menu option Model > KMeans, and specify a key (dataset).

Please specify the desired number of clusters K (8 for this example dataset), and Epsilon, the termination parameter. When none of the clusters has moved more than Epsilon on any dimension between two iterations, the algorithm stops.

The output is a set of clusters which will be displayed after a progress page.
